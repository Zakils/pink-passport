# Pink Passport Brandbook

Estrutura pronta para subir no GitHub Pages.

## Como usar
1. Envie **todos** os arquivos desta pasta para o seu repositório.
2. Mantenha o `index.html` na raiz.
3. Mantenha a pasta `assets/` com os mesmos nomes.
4. No GitHub, vá em **Settings > Pages** e publique a branch principal (`main`) na pasta `/root`.

- `index.html`
- `assets/`
